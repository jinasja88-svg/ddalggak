# 1688 소싱 플랫폼 사업 계획

> 작성일: 2026년 3월 4일

---

## 한 줄 요약

> **중국 최대 도매 사이트 1688의 상품을, 우리 앱에서 쉽게 검색하고 대신 구매해 배송까지 해주는 서비스입니다.**

---

## 이게 뭔가요? (쉽게 설명)

중국에는 **1688**이라는 아주 큰 도매 쇼핑몰이 있어요.
거기엔 엄청나게 저렴한 상품이 수백만 개 있지만, 중국어라서 쓰기가 어렵고, 직접 주문하기도 복잡해요.

우리가 만들 서비스는 이렇습니다:

```
고객이 우리 앱에서 "가방" 검색
    → 1688의 가방 수백 개가 한국어로 뜸
    → 고객이 마음에 드는 거 골라서 주문 + 결제
    → 우리가 대신 중국에서 사서 국내로 배송
```

쉽게 말하면 **"중국 직구 대신해주는 서비스"** 입니다.

---

## 왜 이 사업이 좋은가요?

| 이유 | 설명 |
|------|------|
| **시장이 큼** | 중국 직구 시장은 매년 30% 성장 중 |
| **이미 잘 되는 서비스 있음** | 소싱라이프(sellochomes) 등 경쟁 서비스가 이미 수익 내는 중 |
| **기존 자산 재활용 가능** | 딸깍러 앱이 이미 있어서 처음부터 만들 필요 없음 → 개발 비용 절감 |
| **수익 구조 명확** | 상품가의 10~15% 수수료 수입 |

---

## 어떻게 작동하나요?

```
[1단계] 고객이 우리 앱에서 상품 검색
         ↓
[2단계] 우리 서버가 1688에서 상품 정보를 가져와 한국어로 번역해서 보여줌
         ↓
[3단계] 고객이 주문 + 카드 결제 (토스페이먼츠)
         ↓
[4단계] 우리 직원(또는 자동화)이 1688에서 실제로 구매
         ↓
[5단계] 중국 → 한국 배송, 고객에게 운송장 번호 제공
         ↓
[6단계] 배송 완료
```

---

## 핵심 기능 4가지

### 1. 상품 검색
- 한국어로 검색하면 자동 번역 후 1688에서 상품을 가져옴
- 카테고리별 탐색 가능 (의류, 가방, 전자제품 등)

### 2. 실시간 가격 계산
- 중국 위안(¥) 가격을 실시간 환율로 원화(₩)로 자동 변환
- 수수료 + 배송비 포함한 최종 금액 미리 보여줌

**예시:**
```
1688 상품가:  ¥50 → 약 9,250원
수입대행 수수료: 약 1,400원 (15%)
배송비:       약 5,000원
─────────────────────────
고객 최종 부담: 약 15,650원
```

### 3. 이미지로 검색 (나중에 추가)
- 사진을 올리면 비슷한 상품을 1688에서 찾아줌
- "이거랑 똑같은 거 중국에서 더 싸게 살 수 있나요?" 용도

### 4. 주문 추적
- 주문 후 배송 현황을 앱에서 실시간으로 확인 가능

---

## 개발 일정 (총 8주)

```
1~2주차 (Phase 1)  : 1688 연결 + 상품 검색 화면 만들기
3~4주차 (Phase 2)  : 주문 + 결제 기능 붙이기
5~6주차 (Phase 3)  : 배송 추적 + 가격 자동 계산
7~8주차 (Phase 4)  : 관리자 화면 + 성능 개선
```

## 돈이 얼마나 드나요?

### 월 운영 비용

| 항목 | 비용 | 설명 |
|------|------|------|
| 1688 데이터 이용료 | 약 13~40만원/월 | 상품 정보 가져오는 API |
| 서버/데이터베이스 | 무료~약 3만원/월 | 현재 쓰는 것 그대로 활용 |
| 기타 (환율, 번역) | 거의 무료 | 무료 플랜으로 충분 |
| **합계** | **약 14~44만원/월** | |

### 개발 비용
- 기존 딸깍러 앱 위에 기능을 추가하는 방식이라 **새로 만드는 것보다 훨씬 저렴**
- 외주 대비 60~70% 절감 가능

---

## 수익은 어떻게 나나요?

```
고객 결제금액의 10~15%가 우리 수익

예) 고객이 한 달에 100만원어치 주문
    → 우리 수익: 10~15만원

고객 100명이 각 100만원씩 주문
    → 월 수익: 1,000~1,500만원
```

운영 비용(최대 44만원)을 제외하면 **손익분기점이 매우 낮습니다.**

---

## 법적으로 문제없나요?

| 항목 | 내용 |
|------|------|
| **사업자 등록** | 통신판매업 신고 필요 (간단히 가능) |
| **관세** | 150달러 이하 소액 구매는 관세 없음 (소액면세 제도) |
| **금지 품목** | 식품, 의약품, 전기용품 인증 미비 제품은 취급 제외 |
